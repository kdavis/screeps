# Snez's Screeps Repository

This repository is holding my "artificial intelligence" and aims to be fully automated.

## Progression

Each "room" should be considered it's own entity. In the event of an "emergency" or an attack, there should be methods to request help from other rooms. 
In the event of a room being overpowered by an invasion, the room should prioritise extracting resources to safer rooms.

### Creep Roles

|Role|Description|
|----|-----------|
|Roadworker|This creep should be designed to *only* build roads between two places of interest. If available, to build a dual lane road to allow bi-directional passage without slowing down creeps.|
|Harvester|This creep should gather resources or energy from a source and deposit it into a container for storage|
